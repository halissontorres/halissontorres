## Projetos em Destaque

### [Ajuste de Correlação Hindemburg](https://github.com/halissontorres/ajuste-correlacao-hindemburg)
Implementação da fórmula HM de Hindemburg Melão Jr para correção da correlação de Pearson em casos com restrição de faixa, onde a correlação observada pode ser subestimada. Inclui serviço REST com Docker.

**Stack:** Python, Flask, Docker, Estatística

### [HM Correlation Visualizer](https://github.com/halissontorres/hm-correlation-visualizer)
Interface web para visualização de correlações corrigidas pelo Método Hindemburg. Permite upload de arquivos CSV, seleção de colunas, aplicação de filtros e visualização gráfica dos resultados.

**Stack:** Next.js, TypeScript, Tailwind CSS, Docker

### [Markov Chain](https://github.com/halissontorres/markov-chain)
Implementação de Cadeias de Markov em Java com Quarkus. Utiliza padrões Strategy e Builder para simulação de processos estocásticos.

**Stack:** Java, Quarkus

### [Pós-PUC FullStack](https://github.com/halissontorres/pos-puc-fullstack-dev)
Repositório acadêmico com projetos desenvolvidos durante a Pós-Graduação em FullStack Development — PUC-RS. Inclui trabalhos de algoritmos, POO, DevOps, bancos de dados, microsserviços e arquitetura em nuvem.

**Stack:** Python, JavaScript, Docker, SQL, NoSQL

### [TJPB Alert](https://github.com/halissontorres/tjpb-alert)
Sistema de notificação de emergências em tempo real para segurança institucional do Tribunal de Justiça da Paraíba. Desenvolvido como Trabalho de Conclusão de Curso - Pós-Graduação em Desenvolvimento Full Stack 

**Stack:** Node.js, express, React, Sequelize ORM, Tailwind CSS
